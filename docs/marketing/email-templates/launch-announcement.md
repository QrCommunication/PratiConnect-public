# Template Email : Annonce Nouvelle Fonctionnalité

> Template réutilisable pour toute annonce de feature

---

## Paramètres du Template

| Champ | Valeur |
|-------|--------|
| **Déclencheur** | Manuel (lancement feature) |
| **Segment** | Tous utilisateurs actifs |

---

## Variables à Personnaliser

| Variable | Description | Exemple |
|----------|-------------|---------|
| `{{feature_name}}` | Nom de la fonctionnalité | Body Mapping 3D |
| `{{feature_benefit}}` | Bénéfice principal | Visualisez les zones traitées |
| `{{feature_description}}` | Description 2-3 phrases | Localisez les points... |
| `{{cta_text}}` | Texte du bouton | Découvrir le Body Mapping |
| `{{cta_url}}` | URL de destination | /app/body-mapping |

## Exemple Complet : Body Mapping 3D

**Sujet** : Nouveau : le Body Mapping 3D est disponible

**Preheader** : Visualisez et annotez les zones du corps sur un modèle interactif

---

Bonjour {{prenom}},

Nous avons une nouvelle fonctionnalité à vous présenter : **le Body Mapping 3D**.

**Visualisez et annotez les zones traitées directement sur un modèle anatomique interactif.**

Que vous soyez ostéopathe, kinésithérapeute, réflexologue ou acupuncteur, vous pouvez maintenant localiser précisément les points de douleur, les zones traitées ou les méridiens sur un modèle 3D. L'historique visuel de chaque séance est conservé, vous permettant de suivre l'évolution dans le temps.

[Découvrir le Body Mapping]

**Comment l'utiliser ?**

1. Ouvrez la fiche d'un patient
2. Cliquez sur l'onglet "Body Mapping"
3. Sélectionnez la zone du corps, annotez, enregistrez

**Pourquoi nous l'avons créé ?**

Plusieurs ostéopathes nous ont demandé un moyen de montrer à leurs patients où ils avaient travaillé pendant la séance. Le Body Mapping répond à ce besoin : une communication claire, un suivi visuel, une meilleure compréhension pour le patient.

Nous avons hâte de connaître votre avis. Répondez à cet email pour nous dire ce que vous en pensez.

L'équipe PratiConnect
