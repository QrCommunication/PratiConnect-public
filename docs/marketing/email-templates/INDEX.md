## Conventions de Rédaction

### Ton et Style

- **Vouvoiement** obligatoire
- Ton professionnel mais chaleureux
- Phrases courtes (max 20 mots)
- Paragraphes courts (max 3 lignes)

### Structure Standard

1. **Sujet** : Max 50 caractères, personnalisé si possible
2. **Preheader** : Complément du sujet, max 80 caractères
3. **Salutation** : "Bonjour [Prénom],"
4. **Corps** : 150-250 mots max
5. **CTA** : 1 seul CTA principal, texte d'action
6. **Signature** : L'équipe PratiConnect

### Variables de Personnalisation

| Variable | Description | Exemple |
|----------|-------------|---------|
| `{{prenom}}` | Prénom du destinataire | Sophie |
| `{{specialite}}` | Spécialité choisie | naturopathe |
| `{{jours_essai}}` | Jours restants essai | 7 |
| `{{feature}}` | Fonctionnalité mentionnée | questionnaires |
