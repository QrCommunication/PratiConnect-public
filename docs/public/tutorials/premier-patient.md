# Tutoriel : Creer votre premier patient

> Apprenez a ajouter un patient et configurer sa fiche complete.

**Duree** : 5 minutes

**Prerequis** : Avoir un compte PratiConnect actif

---

## Ce que vous allez apprendre

- Creer une fiche patient
- Renseigner les informations de contact
- Ajouter les informations medicales
- Organiser avec des tags

---

## Etape 1 : Acceder a la creation de patient

1. Connectez-vous a votre compte PratiConnect
2. Dans le menu lateral gauche, selectionnez **Patients**
3. Selectionnez le bouton **+ Nouveau patient** en haut a droite

![Bouton nouveau patient](../../assets/screenshots/btn-nouveau-patient.png)

**Resultat** : Une fenetre de creation s'ouvre avec un formulaire en 4 etapes.

---

## Etape 2 : Renseigner l'identite

Remplissez les champs suivants :

| Champ | Obligatoire | Exemple |
|-------|-------------|---------|
| Civilite | Oui | Mme |
| Prenom | Oui | Marie |
| Nom | Oui | Dupont |
| Date de naissance | Oui | 15/03/1985 |
| Genre | Non | Feminin |

Selectionnez **Suivant** pour passer a l'etape suivante.

> **Astuce** : La date de naissance permet de calculer automatiquement l'age du patient.

---

## Etape 3 : Ajouter les coordonnees

Renseignez les informations de contact :

**Telephone mobile** (recommande)
- Format : 06 12 34 56 78
- Utilise pour les rappels SMS

**Email**
- Necessaire pour le portail patient
- Permet l'envoi de questionnaires et factures

**Adresse postale**
- Numero et rue
- Code postal
- Ville
- Pays

Selectionnez **Suivant**.

> **Note** : Le telephone mobile est important pour les rappels de rendez-vous.

---

## Etape 4 : Completer les informations medicales (optionnel)

Cette etape peut etre completee plus tard, mais il est utile de noter les informations essentielles des maintenant.

**Informations permanentes**
- Groupe sanguin
- Taille (cm)
- Poids (kg)

**Allergies connues**

1. Selectionnez **+ Ajouter une allergie**
2. Indiquez le nom de l'allergene
3. Choisissez le niveau de severite :
   - Legere (rhinite, demangeaisons)
   - Moderee (urticaire, gonflement)
   - Severe (choc anaphylactique)
4. Decrivez la reaction

**Exemple** :
- Allergene : Penicilline
- Severite : Severe
- Reaction : Choc anaphylactique

**Traitements en cours**

1. Selectionnez **+ Ajouter un traitement**
2. Indiquez le nom du medicament
3. Precisez la posologie (ex: 1 comprime le matin)

Selectionnez **Suivant**.

---

## Etape 5 : Configurer les options

**Tags personnalises**

Les tags vous aident a organiser vos patients. Exemples :
- Osteo
- Suivi mensuel
- Entreprise X

Pour ajouter un tag :
1. Tapez le nom du tag
2. Selectionnez-le dans la liste ou creez-en un nouveau

**Source de referencement**

Comment le patient vous a-t-il connu ?
- Bouche-a-oreille
- Site web
- Reseaux sociaux
- Annuaire en ligne
- Autre

> **Astuce** : Suivre vos sources de referencement vous aide a comprendre d'ou viennent vos patients.

**Consentement RGPD**

Cochez la case pour confirmer que le patient consent au traitement de ses donnees de sante. Cette case est obligatoire.

PratiConnect enregistre automatiquement :
- La date du consentement
- L'heure
- La version des CGU acceptees

---

## Etape 6 : Valider la creation

1. Verifiez les informations dans le recapitulatif
2. Selectionnez **Creer le patient**

**Resultat** : La fiche patient s'ouvre. Vous pouvez maintenant :
- Planifier un rendez-vous
- Demarrer une session
- Envoyer un questionnaire

![Fiche patient creee](../../assets/screenshots/fiche-patient-creee.png)

---

## Completer la fiche plus tard

Vous n'etes pas oblige de tout remplir immediatement. Vous pouvez completer la fiche au fil du temps :

1. Ouvrez la fiche du patient
2. Selectionnez l'onglet correspondant (Medical, Contact...)
3. Modifiez les informations
4. Les modifications sont enregistrees automatiquement

---

## Bonnes pratiques

| Pratique | Benefice |
|----------|----------|
| Toujours verifier l'email | Permet le portail patient et les notifications |
| Noter les allergies des le debut | Securite du patient |
| Utiliser des tags coherents | Retrouver facilement les patients |
| Demander la source de referencement | Mesurer l'efficacite de votre communication |

---

## En cas de probleme

### Le patient existe deja

Si l'email ou le telephone est deja associe a un patient, un message vous alerte. Vous pouvez :
- Ouvrir la fiche existante
- Modifier les informations

### J'ai fait une erreur

Toutes les informations peuvent etre modifiees apres la creation :
1. Ouvrez la fiche patient
2. Selectionnez l'onglet concerne
3. Modifiez le champ
4. Les modifications sont sauvegardees automatiquement
