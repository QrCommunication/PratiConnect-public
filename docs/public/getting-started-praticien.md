## 1. Creer votre compte

### Pourquoi cette etape ?

Votre compte PratiConnect est la porte d'entree vers tous les outils de gestion de votre cabinet. Il est securise et conforme aux normes de protection des donnees de sante.

### Comment faire

1. Rendez-vous sur [praticonnect.com](https://praticonnect.com)
2. Selectionnez "Commencer gratuitement"
3. Renseignez votre adresse email professionnelle
4. Choisissez un mot de passe robuste (minimum 12 caracteres)
5. Validez votre email en cliquant sur le lien recu

> **Astuce** : Utilisez une adresse email que vous consultez regulierement. Toutes les notifications importantes y seront envoyees.

### Resultat attendu

Vous accedez a votre tableau de bord vierge, pret a etre configure.

![Tableau de bord initial](../assets/screenshots/dashboard-initial.png)

## 2. Completer votre profil professionnel

### Pourquoi cette etape ?

Votre profil apparait sur vos factures, vos documents et votre page publique. Des informations completes renforcent votre credibilite aupres de vos patients.

### Comment faire

1. Dans le menu lateral, selectionnez "Parametres"
2. Ouvrez l'onglet "Profil professionnel"
3. Remplissez les informations suivantes :

**Identite professionnelle**
- Nom et prenom
- Photo de profil (recommande : format carre, minimum 400x400 pixels)
- Specialite principale (osteopathe, naturopathe, psychotherapeute...)
- Numero ADELI ou RPPS (si applicable)

**Coordonnees du cabinet**
- Adresse complete
- Telephone professionnel
- Site web (optionnel)

**Informations legales**
- Numero SIRET
- Regime fiscal (micro-entreprise, EI, EURL...)
- Mentions legales pour vos factures

4. Selectionnez "Enregistrer les modifications"

> **Attention** : Le numero SIRET est obligatoire pour generer des factures conformes.

### Resultat attendu

Votre profil affiche la mention "Profil complet" avec une coche verte.

![Profil complet](../assets/screenshots/profil-complet.png)

## 3. Configurer vos horaires

### Pourquoi cette etape ?

Vos horaires determinent les creneaux disponibles pour la prise de rendez-vous en ligne. Une configuration precise evite les conflits d'agenda.

### Comment faire

1. Allez dans "Parametres" puis "Agenda"
2. Dans la section "Horaires d'ouverture", definissez vos disponibilites :

**Pour chaque jour de la semaine :**
- Cochez le jour si vous travaillez
- Indiquez l'heure de debut et de fin
- Ajoutez une pause si necessaire (dejeuner, par exemple)

**Exemple de configuration :**
- Lundi : 9h00 - 12h30, 14h00 - 19h00
- Mardi : 9h00 - 12h30, 14h00 - 19h00
- Mercredi : Ferme
- Jeudi : 9h00 - 12h30, 14h00 - 19h00
- Vendredi : 9h00 - 17h00

3. Configurez les types de rendez-vous :

| Type | Duree | Tarif |
|------|-------|-------|
| Premiere consultation | 60 min | 70 EUR |
| Suivi | 45 min | 55 EUR |
| Bilan annuel | 90 min | 100 EUR |

4. Selectionnez "Enregistrer"

> **Astuce** : Vous pouvez creer des creneaux specifiques pour la teleconsultation avec des tarifs differents.

### Synchronisation Google Calendar (optionnel)

Pour eviter les doublons avec votre agenda personnel :

1. Dans "Parametres" puis "Integrations"
2. Selectionnez "Connecter Google Calendar"
3. Autorisez l'acces a votre calendrier
4. Choisissez le calendrier a synchroniser

Les evenements de votre Google Calendar apparaitront comme "indisponibles" dans PratiConnect.

### Resultat attendu

Votre agenda affiche les creneaux disponibles selon vos horaires. Les jours fermes apparaissent grises.

![Configuration agenda](../assets/screenshots/agenda-configuration.png)

## 4. Ajouter votre premier patient

### Pourquoi cette etape ?

Chaque patient dispose d'un dossier complet dans PratiConnect : coordonnees, historique medical, sessions passees, documents. Prendre quelques minutes pour bien renseigner le dossier vous fera gagner du temps par la suite.

### Comment faire

1. Dans le menu lateral, selectionnez "Patients"
2. Selectionnez le bouton "+ Nouveau patient"
3. Remplissez le formulaire en 4 etapes :

**Etape 1 : Identite**
- Civilite, nom, prenom
- Date de naissance
- Genre

**Etape 2 : Contact**
- Telephone mobile (obligatoire pour les rappels SMS)
- Adresse email
- Adresse postale

**Etape 3 : Informations medicales** (optionnel a ce stade)
- Groupe sanguin
- Allergies connues
- Traitements en cours

**Etape 4 : Options**
- Tags personnalises (exemple : "osteo", "suivi mensuel")
- Source de referencement (bouche-a-oreille, site web...)
- Consentement RGPD

4. Selectionnez "Creer le patient"

> **Note** : Le consentement RGPD est obligatoire pour traiter les donnees de sante. PratiConnect enregistre automatiquement la date et l'heure du consentement.

### Resultat attendu

La fiche patient s'ouvre avec toutes les informations renseignees. Vous pouvez maintenant planifier sa premiere session.

![Fiche patient](../assets/screenshots/fiche-patient.png)

## 5. Realiser votre premiere session

### Pourquoi cette etape ?

Une session dans PratiConnect correspond a une consultation avec un patient. Elle regroupe vos notes, les mesures prises, les actes realises et la signature du patient.

### Comment faire

**Planifier la session :**

1. Depuis la fiche patient, selectionnez "Nouvelle session"
2. Choisissez la date et l'heure
3. Selectionnez le type de session (premiere consultation, suivi...)
4. Confirmez la creation

**Demarrer la session le jour J :**

1. Depuis votre tableau de bord, retrouvez le rendez-vous du jour
2. Selectionnez "Demarrer" pour lancer la session
3. Le chronometre demarre automatiquement

**Pendant la session :**

Utilisez les differents onglets selon vos besoins :

| Onglet | Utilisation |
|--------|-------------|
| Notes | Rediger vos observations cliniques |
| Vitaux | Enregistrer les constantes (tension, poids...) |
| Anamnese | Remplir le bilan initial |
| Body mapping | Annoter les zones sur un schema corporel |
| Questionnaires | Faire remplir une echelle au patient |
| Actes | Ajouter les actes realises |

**Terminer la session :**

1. Selectionnez l'onglet "Resume"
2. Verifiez les actes et le montant total
3. Demandez au patient de signer sur l'ecran ou la tablette
4. Selectionnez "Terminer la session"

> **Astuce** : Les notes sont sauvegardees automatiquement toutes les 30 secondes. Pas besoin de cliquer sur "Enregistrer".

### Resultat attendu

La session apparait dans l'historique du patient avec le statut "Terminee". Vous pouvez desormais generer une facture.

![Session terminee](../assets/screenshots/session-terminee.png)

## 6. Configurer la facturation

### Pourquoi cette etape ?

Une facturation bien configuree vous permet d'encaisser rapidement et de generer des documents conformes pour vos patients et votre comptabilite.

### Comment faire

**Configurer l'identite de facturation :**

1. Allez dans "Parametres" puis "Facturation"
2. Verifiez vos informations legales (SIRET, adresse...)
3. Ajoutez vos coordonnees bancaires (pour les virements)
4. Personnalisez le modele de facture :
   - Logo
   - Mentions legales
   - Conditions de paiement

**Configurer les modes de paiement :**

PratiConnect propose plusieurs options :

| Mode | Configuration |
|------|---------------|
| Especes | Aucune configuration requise |
| Cheque | Aucune configuration requise |
| Virement | Ajoutez votre IBAN |
| Carte bancaire | Connectez Stripe |
| Terminal de paiement | Connectez SumUp (France uniquement) |

**Connecter Stripe (paiement par carte) :**

1. Dans "Parametres" puis "Integrations"
2. Selectionnez "Connecter Stripe"
3. Suivez les etapes de creation de compte Stripe
4. Une fois connecte, le statut passe a "Actif"

**Creer votre premiere facture :**

1. Depuis la session terminee, selectionnez "Facturer"
2. Verifiez les lignes de facturation
3. Choisissez le mode de paiement
4. Selectionnez "Generer la facture"

> **Note** : Si vous n'etes pas assujetti a la TVA, la mention legale appropriee s'affiche automatiquement.

### Resultat attendu

Votre facture est generee avec un numero unique (format INV-2026-00001). Vous pouvez l'envoyer par email ou l'imprimer.

![Facture generee](../assets/screenshots/facture-generee.png)

## 7. Activer le portail patient

### Pourquoi cette etape ?

Le portail patient offre un espace en ligne ou vos patients peuvent prendre rendez-vous, remplir des questionnaires, consulter leurs documents et payer leurs factures. Cela reduit le temps passe au telephone et ameliore l'experience patient.

### Comment faire

**Activer le portail pour un patient :**

1. Ouvrez la fiche du patient
2. Dans la section "Portail patient", selectionnez "Activer l'acces"
3. Un email d'invitation est envoye automatiquement

**Ce que recoit le patient :**

Un email contenant :
- Un lien de connexion securise (valide 24 heures)
- Les instructions pour definir son mot de passe
- Les fonctionnalites disponibles

**Personnaliser les autorisations :**

Dans "Parametres" puis "Portail patient", definissez ce que les patients peuvent faire :

| Option | Recommandation |
|--------|----------------|
| Prendre rendez-vous en ligne | Active |
| Annuler un rendez-vous | Active (avec delai minimum) |
| Consulter les documents partages | Active |
| Payer les factures en ligne | Active si Stripe est configure |
| Acceder aux medias therapeutiques | Selon votre pratique |

### Tester le portail

Avant d'inviter vos patients, testez le portail vous-meme :

1. Creez un patient fictif avec votre adresse email personnelle
2. Activez son acces au portail
3. Connectez-vous au portail pour verifier l'experience

### Resultat attendu

Votre patient recoit son invitation et peut se connecter de maniere autonome a son espace.

![Portail patient](../assets/screenshots/portail-patient-invitation.png)

## En cas de probleme

### Je n'ai pas recu l'email de validation

1. Verifiez votre dossier de courriers indesirables (spam)
2. Attendez quelques minutes, le delai peut aller jusqu'a 5 minutes
3. Selectionnez "Renvoyer l'email de validation" sur la page de connexion

### Je ne trouve pas une fonctionnalite

La navigation principale se trouve dans le menu lateral gauche :
- **Tableau de bord** : Vue d'ensemble de votre journee
- **Patients** : Liste et dossiers patients
- **Agenda** : Calendrier et rendez-vous
- **Sessions** : Consultations en cours et passees
- **Facturation** : Factures et paiements
- **Parametres** : Configuration du compte

### L'interface est en anglais

1. Allez dans "Settings" (Parametres)
2. Selectionnez "Language" (Langue)
3. Choisissez "Francais"
