## Compte et inscription

### Comment creer mon compte PratiConnect ?

Rendez-vous sur [praticonnect.com](https://praticonnect.com) et selectionnez "Commencer gratuitement". Vous devrez renseigner votre email professionnel et valider votre compte via le lien recu par email.

### J'ai oublie mon mot de passe, que faire ?

1. Sur la page de connexion, selectionnez "Mot de passe oublie"
2. Entrez votre adresse email
3. Consultez votre boite de reception (et le dossier spam)
4. Selectionnez le lien pour definir un nouveau mot de passe

### Puis-je utiliser PratiConnect sur plusieurs appareils ?

Oui. Votre compte fonctionne sur ordinateur, tablette et smartphone. Vos donnees sont synchronisees automatiquement.

### Comment modifier mes informations personnelles ?

Allez dans "Parametres" puis "Profil professionnel". Modifiez les champs souhaites et selectionnez "Enregistrer".

### Comment resilier mon abonnement ?

Contactez le support a support@praticonnect.com avec votre demande de resiliation. Vos donnees seront conservees pendant 30 jours avant suppression definitive, conformement au RGPD.

## Gestion des patients

### Comment ajouter un nouveau patient ?

1. Selectionnez "Patients" dans le menu
2. Selectionnez "+ Nouveau patient"
3. Remplissez les informations requises
4. Selectionnez "Creer le patient"

### Puis-je importer mes patients depuis un autre logiciel ? {#import-patients}

Oui. Dans "Parametres" puis "Import/Export", vous pouvez importer un fichier CSV contenant vos patients. Un modele de fichier est fourni pour vous aider a formater vos donnees.

### Comment fusionner deux fiches patient en double ?

Contactez le support avec les identifiants des deux fiches. Cette operation est irreversible et doit etre validee par notre equipe pour eviter toute perte de donnees.

### Comment supprimer un patient ?

1. Ouvrez la fiche du patient
2. Selectionnez les trois points (menu)
3. Selectionnez "Supprimer le patient"
4. Confirmez la suppression

> **Note** : La suppression est definitive. Pour conserver un historique, preferez l'archivage.

### Comment partager un dossier patient avec un collegue ?

1. Ouvrez la fiche du patient
2. Selectionnez "Partager"
3. Entrez l'email du collegue (qui doit avoir un compte PratiConnect)
4. Definissez les droits (lecture seule ou modification)
5. Confirmez le partage

### Comment respecter le droit a l'oubli (RGPD) ?

1. Ouvrez la fiche du patient
2. Selectionnez "Actions RGPD"
3. Selectionnez "Anonymiser le dossier"
4. Confirmez l'operation

Les donnees identifiantes sont supprimees mais l'historique anonymise est conserve pour votre comptabilite.

## Sessions de soin

### Quelle est la difference entre une session et un rendez-vous ?

Un **rendez-vous** est une reservation dans votre agenda. Une **session** est la consultation elle-meme avec les notes, actes et signature.

### Comment demarrer une session ?

1. Depuis votre tableau de bord, trouvez le rendez-vous du jour
2. Selectionnez "Demarrer"
3. Le chronometre se lance automatiquement

### Mes notes sont-elles sauvegardees automatiquement ?

Oui. PratiConnect sauvegarde vos notes toutes les 30 secondes. Un indicateur "Sauvegarde..." apparait brievement.

### Comment ajouter des constantes vitales ?

1. Pendant la session, allez dans l'onglet "Vitaux"
2. Entrez les valeurs mesurees (tension, poids, temperature...)
3. Les donnees sont enregistrees automatiquement

### Comment faire signer le patient sur ma tablette ?

1. A la fin de la session, selectionnez "Terminer"
2. Verifiez le recapitulatif avec le patient
3. Presentez-lui la zone de signature
4. Le patient signe avec son doigt sur l'ecran tactile
5. Selectionnez "Valider la signature"

### Comment modifier une session terminee ?

Vous pouvez completer les notes d'une session terminee :
1. Allez dans "Sessions"
2. Ouvrez la session concernee
3. Modifiez les notes
4. Les modifications sont tracees dans l'historique

> **Attention** : Les actes et la signature ne peuvent pas etre modifies apres validation.

## Agenda et rendez-vous

### Comment synchroniser mon agenda Google Calendar ?

1. Allez dans "Parametres" puis "Integrations"
2. Selectionnez "Connecter Google Calendar"
3. Autorisez l'acces
4. Choisissez les calendriers a synchroniser

Vos evenements personnels apparaitront comme "indisponibles" dans PratiConnect.

### Comment bloquer une plage horaire ?

1. Dans l'agenda, selectionnez la plage a bloquer
2. Selectionnez "Bloquer"
3. Ajoutez un motif (optionnel)
4. Confirmez

### Comment creer un rendez-vous recurrent ?

1. Creez un rendez-vous normalement
2. Cochez "Rendez-vous recurrent"
3. Definissez la frequence (hebdomadaire, mensuel...)
4. Indiquez la date de fin ou le nombre d'occurrences

### Un patient ne s'est pas presente, que faire ?

1. Ouvrez le rendez-vous
2. Selectionnez "Marquer comme absent"
3. Choisissez si vous souhaitez facturer ou non

Le statut "Absent" est enregistre dans l'historique du patient.

### Comment envoyer des rappels de rendez-vous ?

Les rappels sont envoyes automatiquement :
- 48 heures avant par email
- 2 heures avant par SMS (si le numero est renseigne)

Vous pouvez personnaliser ces delais dans "Parametres" puis "Notifications".

## Facturation et paiements

### Comment generer une facture ?

1. Depuis une session terminee, selectionnez "Facturer"
2. Verifiez les lignes de facturation
3. Selectionnez le mode de paiement
4. Selectionnez "Generer la facture"

### Comment encaisser un paiement par carte ?

**Avec Stripe (en ligne) :**
1. Selectionnez "Encaisser par carte"
2. Le patient entre ses informations
3. Le paiement est confirme instantanement

**Avec SumUp (terminal) :**
1. Selectionnez "Encaisser par terminal"
2. Le montant s'affiche sur votre terminal SumUp
3. Le patient presente sa carte

### Comment envoyer une facture par email ?

1. Ouvrez la facture
2. Selectionnez "Envoyer par email"
3. Verifiez l'adresse du patient
4. Selectionnez "Envoyer"

### Comment creer un forfait de seances ?

1. Allez dans "Facturation" puis "Forfaits"
2. Selectionnez "+ Creer un forfait"
3. Definissez le nombre de seances et le tarif
4. Indiquez la duree de validite
5. Enregistrez

### Comment gerer une facture impayee ?

1. Dans "Facturation", filtrez par "En attente"
2. Selectionnez la facture
3. Vous pouvez :
   - Envoyer une relance par email
   - Marquer comme payee (paiement recu par un autre moyen)
   - Annuler la facture

### Comment exporter mes factures pour mon comptable ?

1. Allez dans "Facturation" puis "Export"
2. Selectionnez la periode
3. Choisissez le format (CSV ou PDF)
4. Selectionnez "Telecharger"

## Teleconsultation

### Quels sont les prerequis techniques pour la teleconsultation ?

- Navigateur web recent (Chrome, Firefox, Safari, Edge)
- Camera et microphone fonctionnels
- Connexion internet stable (minimum 2 Mbps)

### Comment lancer une teleconsultation ?

1. Creez un rendez-vous de type "Teleconsultation"
2. Le jour J, selectionnez "Demarrer la teleconsultation"
3. Autorisez l'acces a votre camera et micro
4. Attendez que le patient rejoigne

### Comment envoyer le lien de connexion au patient ?

Le lien est envoye automatiquement dans l'email de confirmation du rendez-vous. Vous pouvez aussi le renvoyer manuellement depuis la fiche du rendez-vous.

### Puis-je enregistrer la teleconsultation ?

Oui, avec le consentement explicite du patient :
1. Pendant la consultation, selectionnez "Enregistrer"
2. Le patient recoit une demande de consentement
3. S'il accepte, l'enregistrement demarre
4. Le fichier est stocke de maniere securisee

### La connexion du patient est mauvaise, que faire ?

Conseillez au patient de :
- Se rapprocher de sa box internet
- Desactiver la video s'il a une connexion faible
- Utiliser une connexion 4G/5G plutot que le Wi-Fi si necessaire

### Puis-je partager mon ecran pendant la consultation ?

Oui. Selectionnez le bouton "Partager l'ecran" et choisissez la fenetre ou l'ecran a partager.

## Questionnaires

### Comment creer un questionnaire personnalise ?

1. Allez dans "Questionnaires" puis "+ Creer"
2. Glissez-deposez les types de questions
3. Configurez chaque question (texte, options, obligatoire...)
4. Testez l'apercu
5. Selectionnez "Enregistrer"

### Comment envoyer un questionnaire a un patient ?

1. Depuis la fiche patient, selectionnez "Envoyer un questionnaire"
2. Choisissez le questionnaire
3. Le patient recoit un email avec le lien pour repondre

### Comment voir les reponses d'un patient ?

1. Ouvrez la fiche du patient
2. Allez dans l'onglet "Questionnaires"
3. Selectionnez la reponse a consulter

### Puis-je voir l'evolution des reponses dans le temps ?

Oui. Dans l'onglet "Questionnaires" du patient, un graphique montre l'evolution des scores si le questionnaire est configure pour le scoring.

### Comment utiliser les questionnaires pendant une session ?

1. Pendant la session, allez dans l'onglet "Questionnaires"
2. Selectionnez "Faire remplir"
3. Le patient peut repondre sur votre ecran ou sur son telephone

## Documents

### Quels formats de fichiers sont acceptes ?

- Images : JPG, PNG, GIF
- Documents : PDF, DOC, DOCX
- Taille maximale : 10 Mo par fichier

### Comment partager un document avec un patient ?

1. Ouvrez la fiche du patient
2. Allez dans l'onglet "Documents"
3. Selectionnez le document
4. Activez "Visible dans le portail patient"

### Comment generer un compte-rendu de session ?

1. A la fin de la session, selectionnez "Generer le compte-rendu"
2. Choisissez le modele
3. Le document PDF est cree automatiquement

### Comment faire signer un document electroniquement ?

1. Importez le document
2. Selectionnez "Demander une signature"
3. Placez les zones de signature
4. Envoyez au patient
5. Le patient signe depuis son portail

La signature est conforme eIDAS et juridiquement valable.

## Portail patient

### Comment activer le portail pour un patient ?

1. Ouvrez la fiche du patient
2. Verifiez que l'email est renseigne
3. Selectionnez "Activer le portail patient"
4. Le patient recoit une invitation par email

### Que peut faire un patient sur le portail ?

Selon vos parametres :
- Consulter ses rendez-vous
- Prendre rendez-vous en ligne
- Remplir des questionnaires
- Consulter et telecharger ses documents
- Payer ses factures
- Rejoindre une teleconsultation

### Comment desactiver le portail pour un patient ?

1. Ouvrez la fiche du patient
2. Dans "Portail patient", selectionnez "Desactiver"
3. Le patient ne pourra plus se connecter

### Le patient n'a pas recu son invitation, que faire ?

1. Verifiez que l'email du patient est correct
2. Demandez-lui de verifier son dossier spam
3. Renvoyez l'invitation depuis sa fiche

## Securite et confidentialite

### Mes donnees sont-elles securisees ?

Oui. PratiConnect utilise :
- Chiffrement des donnees en transit (HTTPS) et au repos (AES-256)
- Hebergement HDS (Hebergement de Donnees de Sante) en France
- Sauvegardes quotidiennes chiffrees
- Authentification securisee

### Qui peut acceder aux donnees de mes patients ?

Uniquement vous et les collegues avec qui vous partagez explicitement un dossier. L'equipe PratiConnect n'a pas acces a vos donnees patients.

### Comment PratiConnect respecte-t-il le RGPD ?

- Consentement explicite des patients enregistre
- Droit d'acces, de rectification et d'effacement
- Export des donnees sur demande
- Anonymisation possible
- Politique de retention des donnees configurable

### Puis-je activer l'authentification a deux facteurs ?

Oui. Dans "Parametres" puis "Securite", activez l'authentification a deux facteurs. Vous recevrez un code par SMS ou via une application d'authentification.

### Que faire si je suspecte un acces non autorise ?

1. Changez immediatement votre mot de passe
2. Verifiez les sessions actives dans "Parametres" puis "Securite"
3. Deconnectez les sessions suspectes
4. Contactez le support : support@praticonnect.com

## Technique

### Quels navigateurs sont compatibles ?

- Google Chrome (recommande)
- Mozilla Firefox
- Apple Safari
- Microsoft Edge

Utilisez toujours la derniere version de votre navigateur.

### L'application est lente, que faire ?

1. Videz le cache de votre navigateur
2. Verifiez votre connexion internet
3. Fermez les onglets inutilises
4. Essayez un autre navigateur

### Je n'arrive pas a me connecter

1. Verifiez que votre email et mot de passe sont corrects
2. Utilisez "Mot de passe oublie" si necessaire
3. Videz le cache et les cookies de votre navigateur
4. Essayez en navigation privee

### Comment signaler un bug ?

Envoyez un email a support@praticonnect.com avec :
- Une description du probleme
- Les etapes pour le reproduire
- Une capture d'ecran si possible
- Votre navigateur et systeme d'exploitation

### Y a-t-il une application mobile ?

Une application mobile est en cours de developpement. En attendant, PratiConnect est optimise pour une utilisation sur tablette et smartphone via le navigateur.
