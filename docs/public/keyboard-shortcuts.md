# Raccourcis clavier

> Gagnez du temps avec les raccourcis clavier de PratiConnect.

---

## Navigation generale

| Raccourci | Action |
|-----------|--------|
| `G` puis `D` | Aller au tableau de bord (Dashboard) |
| `G` puis `P` | Aller aux patients |
| `G` puis `A` | Aller a l'agenda |
| `G` puis `S` | Aller aux sessions |
| `G` puis `F` | Aller a la facturation |
| `G` puis `Q` | Aller aux questionnaires |
| `G` puis `E` | Aller aux parametres (Settings) |

> **Note** : Appuyez sur `G`, relâchez, puis appuyez sur la deuxieme touche.

---

## Recherche

| Raccourci | Action |
|-----------|--------|
| `/` ou `Ctrl` + `K` | Ouvrir la recherche rapide |
| `Echap` | Fermer la recherche |

La recherche rapide permet de trouver :
- Patients (par nom, email, telephone)
- Sessions
- Documents
- Fonctionnalites

---

## Actions globales

| Raccourci | Action |
|-----------|--------|
| `N` puis `P` | Nouveau patient |
| `N` puis `S` | Nouvelle session |
| `N` puis `R` | Nouveau rendez-vous |
| `N` puis `F` | Nouvelle facture |
| `N` puis `Q` | Nouveau questionnaire |

---

## Liste des patients

| Raccourci | Action |
|-----------|--------|
| `Fleche haut/bas` | Naviguer dans la liste |
| `Entree` | Ouvrir la fiche patient |
| `E` | Modifier le patient selectionne |
| `D` | Supprimer le patient selectionne |

---

## Session de soin

| Raccourci | Action |
|-----------|--------|
| `Ctrl` + `S` | Sauvegarder les notes |
| `Alt` + `1` | Onglet Resume |
| `Alt` + `2` | Onglet Notes |
| `Alt` + `3` | Onglet Vitaux |
| `Alt` + `4` | Onglet Questionnaires |
| `Alt` + `5` | Onglet Body mapping |
| `Ctrl` + `Enter` | Terminer la session |

## Modales et formulaires

| Raccourci | Action |
|-----------|--------|
| `Echap` | Fermer la modale |
| `Tab` | Champ suivant |
| `Shift` + `Tab` | Champ precedent |
| `Entree` | Valider le formulaire |

---

## Raccourcis Mac

Sur Mac, remplacez :
- `Ctrl` par `Cmd`
- `Alt` par `Option`

| Windows/Linux | Mac |
|---------------|-----|
| `Ctrl` + `S` | `Cmd` + `S` |
| `Ctrl` + `K` | `Cmd` + `K` |
| `Alt` + `1` | `Option` + `1` |

---

## Personnaliser les raccourcis

Les raccourcis ne sont pas personnalisables pour le moment. Cette fonctionnalite est prevue dans une prochaine version.

---

## Desactiver les raccourcis

Si les raccourcis interferent avec votre utilisation :
1. Allez dans **Parametres** > **Accessibilite**
2. Desactivez "Raccourcis clavier"
