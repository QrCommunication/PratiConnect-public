# Keyboard shortcuts

On Mac, replace `Ctrl` with `Cmd` and `Alt` with `Option`.

---

## Navigation

| Shortcut | Action | Context |
|----------|--------|---------|
| `G` then `D` | Go to dashboard | Global |
| `G` then `P` | Go to patients | Global |
| `G` then `A` | Go to calendar | Global |
| `G` then `S` | Go to sessions | Global |
| `G` then `F` | Go to billing | Global |
| `G` then `Q` | Go to questionnaires | Global |
| `G` then `E` | Go to settings | Global |
| `/` or `Ctrl` + `K` | Open quick search | Global |
| `Escape` | Close search / modal | Global |

Press `G`, release, then press the second key.

|----------|--------|---------|

---

To disable shortcuts: **Settings** > **Accessibility** > uncheck "Keyboard shortcuts".
