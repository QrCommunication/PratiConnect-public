## Report a problem

If you run into an issue:

1. Open an issue on the [GitHub repository](https://github.com/praticonnect/praticonnect/issues)
2. Describe the steps to reproduce the problem
3. Include your browser, operating system, and the page where the issue occurred
4. Attach a screenshot if possible

For urgent questions, email **support@praticonnect.com**.
