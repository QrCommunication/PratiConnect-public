# Déclaration d'accessibilité / Accessibility Statement / הצהרת נגישות

**PratiConnect** s'engage à rendre son service accessible conformément au RGAA 4.1 et aux WCAG 2.1 niveau AA.

## État de conformité / Compliance Status

**Partiellement conforme** — PratiConnect est en cours de mise en conformité avec le RGAA 4.1.

Dernière évaluation : Février 2026

### Résultats des tests

| Critère | Score | Statut |
|---------|-------|--------|
| Images (alt text) | 85% | En cours |
| Navigation clavier | 90% | Conforme |
| Focus visible | 90% | Conforme |
| Contrastes | 80% | En cours |
| Formulaires | 85% | En cours |
| Sous-titres vidéo | 50% | En cours |

### Score global estimé : **71/100**

## Contenus non accessibles

### Non-conformités

1. **Images décoratives** : Certaines images dans les pages d'atterrissage manquent d'attributs alt descriptifs (en cours de correction)
2. **Sous-titres vidéo** : La téléconsultation LiveKit ne dispose pas encore de sous-titres en temps réel
3. **Contrastes** : Quelques éléments d'interface ont un ratio de contraste insuffisant en mode clair

### Dérogations

- Contenus tiers (Stripe checkout, Google Maps) non contrôlés par PratiConnect

### Contenus non soumis

- Documents PDF générés par des tiers (factures Green Invoice)

## Technologies utilisées

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript/TypeScript (React 19)
- WAI-ARIA 1.2
- Radix UI (composants accessibles)

## Environnement de test

- NVDA 2024.x + Firefox
- VoiceOver + Safari (macOS/iOS)
- Clavier seul (navigation Tab/Enter/Escape)

## Droit à la compensation

Si vous rencontrez un défaut d'accessibilité vous empêchant d'accéder à un contenu ou une fonctionnalité, contactez-nous :

- **Email** : accessibilite@praticonnect.com
- **Formulaire** : https://praticonnect.com/contact

Nous nous engageons à vous apporter une réponse dans un délai de 7 jours ouvrés.

## Voies de recours

Si vous constatez un défaut d'accessibilité et que votre signalement n'a pas reçu de réponse satisfaisante, vous pouvez :

- Écrire au Défenseur des droits : https://formulaire.defenseurdesdroits.fr
- Contacter le délégué du Défenseur des droits dans votre région
- Envoyer un courrier : Défenseur des droits, Libre réponse 71120, 75342 Paris CEDEX 07
