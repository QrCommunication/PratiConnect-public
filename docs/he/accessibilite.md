# נגישות

PratiConnect תוכננה להיות שמישה לכולם, ללא קשר ליכולות פיזיות, העדפות או כלי עזר.

אנחנו שואפים לעמידה בתקן WCAG 2.1 רמה AA ו-RGAA 4.1 (מסגרת ההתייחסות הצרפתית לשיפור הנגישות).

---

## תקן מטרה

הממשקים עומדים בקריטריונים של WCAG 2.1 רמה AA:

- יחס ניגודיות מינימלי של 4.5:1 לטקסט רגיל, 3:1 לטקסט גדול
- מבנה כותרות היררכי (H1, H2, H3)
- טקסט חלופי בכל התמונות האינפורמטיביות
- תוויות (labels) משויכות לכל שדה טופס
- תפקידי ARIA לרכיבים אינטראקטיביים
- הכרזות על שינויים בתוכן דינמי

ביקורות נגישות מבוצעות באופן סדיר. ביקורת אחרונה: ינואר 2026.

---

## ניווט במקלדת

PratiConnect שמיש לחלוטין ללא עכבר:

| מקש | פעולה |
| ---- | ----- |
| `Tab` | מעבר לאלמנט הבא |
| `Shift` + `Tab` | חזרה לאלמנט הקודם |
| `Enter` | הפעלת כפתור או קישור |
| `Space` | סימון תיבה או פתיחת תפריט נפתח |
| `חצים` | ניווט ברשימות ותפריטים |
| `Esc` | סגירת חלונות קופצים ותפריטים |

מחוון ויזואלי (focus ring) מסמן בכל עת את האלמנט הפעיל.

קיצורי מקלדת מאפשרים גישה ישירה לפונקציות העיקריות. עיין ב-[רשימת הקיצורים המלאה](raccourcis-clavier.md).

לכיבוי הקיצורים: **הגדרות** > **נגישות** > בטל סימון "קיצורי מקלדת".

## תמיכת RTL (עברית)

PratiConnect תומכת בתצוגה מימין לשמאל בעברית:

- ממשק מלא בכיוון הפוך (ניווט, תפריטים, כפתורים)
- טקסטים מיושרים לימין
- טפסים ורכיבים אינטראקטיביים ממוקמים מחדש

להפעלת עברית: **הגדרות** > **שפה** > בחר "עברית". הממשק מתאים עצמו אוטומטית.

| שפה | כיוון | סטטוס |
| --- | ----- | ------ |
| צרפתית | LTR | זמינה |
| אנגלית | LTR | זמינה |
| עברית | RTL | זמינה |

## התאמה חזותית

### גודל טקסט

הממשק נשאר תקין עם זום דפדפן עד 200%, ללא אובדן מידע.

- Windows/Linux: `Ctrl` + `+` או `Ctrl` + גלגלת
- Mac: `Cmd` + `+` או `Cmd` + גלגלת

גודל הגופן ניתן לכוונון גם ב-**הגדרות** > **נגישות** (קטן, רגיל, גדול, גדול מאוד).

### ניגודיות וצבעים

מידע לעולם אינו מועבר בצבע בלבד:
- שגיאות מלוות באייקונים ובטקסט מסביר
- סטטוסים משתמשים באייקונים ייחודיים בנוסף לצבע
- גרפים כוללים דפוסים או תוויות

### אנימציות

אם מערכת ההפעלה שלך מוגדרת ל-"הפחת אנימציות", PratiConnect מכבה אוטומטית מעברים דקורטיביים, אנימציות טעינה ואפקטי ריחוף מונפשים.

ניתן להפעיל הגדרה זו גם ידנית: **הגדרות** > **נגישות** > "הפחת תנועה".

### טפסים

לכל שדה יש תווית גלויה, הוראות כשנדרש ומחוון חובה. בשגיאה, השדה הרלוונטי מודגש, הודעה מפורשת מתארת את הבעיה והפוקוס עובר לשדה השגוי הראשון.

### זמנים

אף פעולה אינה כפופה לזמן קצוב. במקרה של חוסר פעילות ממושך, אזהרה מופיעה 5 דקות לפני ניתוק אוטומטי, עם אפשרות להארכת החיבור.
