# קיצורי מקלדת

ב-Mac, החלף `Ctrl` ב-`Cmd` ו-`Alt` ב-`Option`.

---

## ניווט

| קיצור | פעולה | הקשר |
| ------ | ----- | ---- |
| `G` ואז `D` | מעבר ללוח הבקרה | גלובלי |
| `G` ואז `P` | מעבר למטופלים | גלובלי |
| `G` ואז `A` | מעבר ליומן | גלובלי |
| `G` ואז `S` | מעבר לפגישות | גלובלי |
| `G` ואז `F` | מעבר לחשבוניות | גלובלי |
| `G` ואז `Q` | מעבר לשאלונים | גלובלי |
| `G` ואז `E` | מעבר להגדרות | גלובלי |
| `/` או `Ctrl` + `K` | פתיחת חיפוש מהיר | גלובלי |
| `Esc` | סגירת חיפוש / חלון | גלובלי |

לחץ על `G`, שחרר, ואז לחץ על המקש השני.
