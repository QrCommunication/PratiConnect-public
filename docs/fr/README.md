## Signaler un problème

Si vous rencontrez un dysfonctionnement :

1. Ouvrez une issue sur le [dépôt GitHub](https://github.com/praticonnect/praticonnect/issues)
2. Décrivez les étapes pour reproduire le problème
3. Précisez votre navigateur, votre système d'exploitation et la page concernée
4. Joignez une capture d'écran si possible

Pour les questions urgentes, écrivez à **support@praticonnect.com**.
