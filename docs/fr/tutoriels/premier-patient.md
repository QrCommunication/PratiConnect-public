# Créer votre premier patient

Ce tutoriel vous guide pas à pas pour ajouter un patient dans PratiConnect et compléter sa fiche.

**Durée** : 5 minutes

## Prérequis

- Un compte PratiConnect actif
- Être connecté à votre espace praticien

---

### 1. Ouvrir le formulaire de création

- Dans le menu latéral gauche, sélectionnez **Patients**.
- Cliquez sur le bouton **+ Nouveau patient** en haut à droite de la liste.
- Un formulaire en 4 étapes s'affiche dans une fenêtre modale.

**Résultat** : le formulaire de création s'ouvre sur l'étape "Identité".

---

### 2. Renseigner l'identité du patient

Remplissez les champs suivants :

| Champ | Obligatoire | Exemple |
|-------|:-----------:|---------|
| Civilité | Oui | Mme |
| Prénom | Oui | Marie |
| Nom | Oui | Dupont |
| Date de naissance | Oui | 15/03/1985 |
| Genre | Non | Féminin |

- La date de naissance sert à calculer l'âge automatiquement dans la fiche.
- Cliquez sur **Suivant** pour passer à l'étape suivante.

**Résultat** : l'étape "Coordonnées" s'affiche.

---

### 3. Ajouter les coordonnées

- **Téléphone mobile** : format 06 12 34 56 78. Ce numéro sert pour les rappels SMS de rendez-vous.
- **Email** : nécessaire pour activer le portail patient, envoyer des questionnaires et des factures.
- **Adresse postale** : numéro, rue, code postal, ville, pays.
- Cliquez sur **Suivant**.

Le téléphone mobile est le canal principal pour les rappels automatiques. Renseignez-le en priorité.

**Résultat** : l'étape "Informations médicales" s'affiche.

---

### 4. Compléter les informations médicales

Cette étape est facultative. Vous pouvez la remplir plus tard depuis la fiche du patient.

**Données permanentes**

- Groupe sanguin, taille (cm), poids (kg).

**Allergies connues**

- Cliquez sur **+ Ajouter une allergie**.
- Saisissez le nom de l'allergène (ex. : Pénicilline).
- Choisissez le niveau de sévérité : Légère, Modérée ou Sévère.
- Décrivez la réaction observée.

**Traitements en cours**

- Cliquez sur **+ Ajouter un traitement**.
- Indiquez le nom du médicament et la posologie (ex. : 1 comprimé le matin).
- Cliquez sur **Suivant**.

**Résultat** : l'étape "Options" s'affiche.

---

### 5. Configurer les options

**Tags personnalisés**

- Tapez un nom de tag dans le champ de recherche (ex. : "Suivi mensuel", "Entreprise X").
- Sélectionnez un tag existant ou créez-en un nouveau.
- Les tags vous aident à filtrer et retrouver vos patients rapidement.

**Source de référencement**

- Indiquez comment le patient vous a trouvé : bouche-à-oreille, site web, réseaux sociaux, annuaire en ligne ou autre.
- Cette information vous permet de mesurer l'efficacité de vos canaux d'acquisition.

**Consentement RGPD**

- Cochez la case de consentement au traitement des données de santé. Cette case est obligatoire.
- PratiConnect enregistre automatiquement la date, l'heure et la version des CGU acceptées.

**Résultat** : le récapitulatif s'affiche avec toutes les informations saisies.

---

### 6. Valider la création

- Relisez le récapitulatif pour vérifier les informations.
- Cliquez sur **Créer le patient**.
- La fiche du patient s'ouvre directement.

**Résultat** : le patient apparaît dans votre liste. Depuis sa fiche, vous pouvez planifier un rendez-vous, démarrer une séance ou envoyer un questionnaire.
