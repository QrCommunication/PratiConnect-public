# Raccourcis clavier

Sur Mac, remplacer `Ctrl` par `Cmd` et `Alt` par `Option`.

---

## Navigation

| Raccourci | Action | Contexte |
|-----------|--------|----------|
| `G` puis `D` | Aller au tableau de bord | Global |
| `G` puis `P` | Aller aux patients | Global |
| `G` puis `A` | Aller à l'agenda | Global |
| `G` puis `S` | Aller aux sessions | Global |
| `G` puis `F` | Aller à la facturation | Global |
| `G` puis `Q` | Aller aux questionnaires | Global |
| `G` puis `E` | Aller aux paramètres | Global |
| `/` ou `Ctrl` + `K` | Ouvrir la recherche rapide | Global |
| `Échap` | Fermer la recherche / modale | Global |

Appuyer sur `G`, relâcher, puis appuyer sur la deuxième touche.
